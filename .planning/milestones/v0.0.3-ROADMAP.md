## Key Decisions

- **Install mechanism RESOLVED:** npm (CLI) + self-hosted in-repo marketplace (skills). No separate repo, no `.zip` build feature.
- **`source: npm` without `version`** — always resolve latest from npm; add pinning only if required.
- **`strict: false`** — marketplace entry is the full plugin definition; `dist/public/.claude-plugin/plugin.json` is 2 levels deep and not found by plugin-root lookup. Confirmed by `claude plugin validate .`.
- **Public plugin name = marketplace name = `motto`** — daily slash-command ergonomics (`/motto:*`) beat install-string readability; schema-valid.
- **Repo visibility:** made public at ship for DOC-02 external-user path, then reverted to **private** (sole-user context). Marketplace add works over maintainer `gh` creds while private; README DOC-02 assumes public for others.
