| npm published tarball → stranger's installed plugin | The content a stranger installs may differ from repo source (stale/defective tarball); the diff-against-source step is the only trustworthy check across this boundary |
| Claude Code plugin cache → what the user actually loads | A stale local cache can mask the real published content; the README caveat and the content diff (not cache status) address this |
